# React Practice Projects 🚀

This repository contains multiple mini projects built while learning React. These projects helped me understand React fundamentals such as components, props, state management, hooks, event handling, API integration, and browser storage.

---

## 📂 Projects Included

### 🌤️ Weather App
A weather application that allows users to search for a city and view real-time weather information.

**Features**
- Search weather by city
- Displays temperature and weather conditions
- Dynamic background based on weather
- Responsive UI

**Concepts Used**
- useState
- useEffect
- API Integration
- Conditional Rendering

---

### 🎬 Movie Search App
A movie searching application that fetches movie information from an external API.

**Features**
- Search movies by title
- Displays movie details and posters
- Real-time data fetching

**Concepts Used**
- API Calls
- State Management
- Conditional Rendering

---

### 💰 Expense Tracker
An application for managing daily expenses.

**Features**
- Add expenses
- View expense list
- Track spending
- Data persists even after page refresh using Local Storage

**Concepts Used**
- useState
- Forms and Inputs
- List Rendering
- Local Storage API
- State Management

---

### ✅ To-Do App
A simple task management application.

**Features**
- Add tasks
- Delete tasks
- Manage daily activities

---

### ❌⭕ Tic-Tac-Toe
A two-player Tic-Tac-Toe game.

**Features**
- Winner detection
- Game reset functionality
- Interactive UI

---

### 🔢 Counter App
A basic counter application.

**Features**
- Increment counter
- Decrement counter
- Reset functionality

---

### ✊✋✌️ Rock Paper Scissors
A game where users can play against the computer.

**Features**
- Random computer choices
- Score tracking
- Interactive gameplay

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/priyaxdev/react-practise.git
```

### Move into the repository

```bash
cd react-practise
```

### Open any project folder

```bash
cd weather-app-v2
# or
cd movie-search-app
# or
cd Expense-tracker
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```
